    LABERINTO

    Aquí otro de nuestros juegos, el juego del laberinto infinito que te hará estallar la cabeza.

    ¡Esperamos que os guste!

    Descripción

    Laberinto es un juego de resolución de laberintos en el que cada vez que superes uno, sumarás un punto y afrontarás un nuevo circuito. Además, sólo tendrás 90 segundos para resolver cada laberinto.

    La dificultad de estos laberintos es variable y aleatoria, al crearse estos con unas dimensiones aleatorias cada vez que tengamos uno nuevo. Cuanto mayor sean las dimensiones, mayor será la dificultad.

    La autoría del proyecto pertenece a Noé Andreu.

    El objetivo de este juego es mostrar un ejemplo de juegos sencillos y divertidos que podemos hacer con Processing.

    Programa

    A continuación, se muestra el código comentado que se ha implementado en Processing para crear el juego.

    Este código está formado por el código general con sus funciones y cuatro clases para controlar los diferentes parámetros.
